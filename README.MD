# 로또 기계

## 📝 프로젝트 소개
Kotlin을 이용하여 로또 머신이 수행하는 로직을 구현한다.

## 기능
- **입력한 값 유효성 검증** : 1~45 범위, 중복 확인, 6개 범호를 체크 한다.
- **자동 번호 생성** : 6개의 서로 다른 번호로 로또를 생성한다.
- **수동 번호 생성** : 사용자에게 입력받은 6개의 번호로 로또를 생성한다.
- **당첨 번호 확인** : 당첨 번호와 보너스 번호를 비교해 일치 하는 숫자를 반환해 준다.
- **당첨 등수 판정** : 일치한 로또 번호들로 당첨된 등수를 판정한다.
- **당첨 금액 & 수익률 계산** : 등수에 따른 당첨 금액과 사용자가 넣은 금액 대비 수익률을 알려준다.
